# HModelGrid
iOS app displaying vertical grid of products.

# Requirements 
* Xcode 16.2
* iOS >= 16.0

## App
Implemented utilizing:
* Swift 6.
* Strict Concurrency Checking = Complete.
* MVVM + SwiftUI.
* Actors for background tasks.
* Async/await.
* Image caching.
* Unit tests.
* Dark mode support.
* Behaviour configured within `HMApp.Configuration` type.
